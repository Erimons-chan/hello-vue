# hello-vue  
**[demo](https://jintangwang.github.io/hello-vue/)**
> vue2.0的demo，学习了elementUI组件，自定义markdown的指令...

## 运行此项目

``` bash
# clone respository
git clone git@github.com:jintangWang/hello-vue.git

# enter the directory 
cd hello-vue

# install dependencies
npm install

# serve with hot reload at localhost:8082
npm run dev

```
